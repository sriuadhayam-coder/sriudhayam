# 🌐 Sri Udhayam Jobs Portal

A simple and responsive job portal website designed to connect job seekers with employers. This platform provides transparent and reliable manpower solutions across multiple industries.

---

## 📌 About the Project

**Sri Udhayam Jobs** is a government-registered consultancy that helps job seekers find suitable employment opportunities and assists companies in fulfilling their manpower requirements efficiently.

---

## 🚀 Features

- 🔍 Job search with filters (by title & category)
- 📂 Multiple job categories (Factory, Farm, Driver, Office, etc.)
- 🖼️ Dynamic job cards with images
- 📱 Fully responsive design (mobile-friendly)
- 📄 Easy application process via Google Form
- 🏢 Employer manpower request form

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS)
